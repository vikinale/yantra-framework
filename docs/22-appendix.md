# 22. Appendix

## 22.1 Glossary
- PSR: PHP Standards Recommendations
- Middleware: request/response pipeline components
- Route cache: precompiled routing tables per HTTP method
- TTL: time-to-live for cache entries

## 22.2 Coding standards
- declare(strict_types=1) in every PHP file
- No @ error suppression
- Avoid superglobals in application code
- Prefer explicit exceptions and central handling
